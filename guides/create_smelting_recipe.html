Create Smelting Recipe

Furnace recipes are simple enough that we don't need to create them in a JSON file.

  1. Navigate to and open /MDKExample/src/main/java/com/digitaladventures/digimod/init/ModRecipes.java
  2. Read the smelting recipes already created in the game.
  3. The addSmelting() method has 4 parts. The Item you create, the Item we use as the ingredient, the total items and the experience gained.
  4. The Item we recieve and the amount are consolidated into an ItemStack object (a stack of a certain item).
    • Input: (Item or Block) The block or item that we place into the furnace.
    • Result: (ItemStack) An object that contains a new instance of an item or block as well as the total(int).
    • Experience: (Float)
  5. Create your own line of code right beneath these smelting recipes and customize the options/argument.
