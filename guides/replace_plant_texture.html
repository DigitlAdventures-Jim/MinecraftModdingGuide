Replace Plant Texture

This is a guide that will help students replace digiplant items textures with their own designs which is the quickest way to have their creations come to life in minecraft.

Replace Plant Texture Files

  1. Name the image files to match digi_plant0-7.
  2. Navigate to MDKExample/src/main/resources/assets.dm/textures/blocks.
  3. Drag your block textures into the blocks folder either on top of any of the files in the folder, or on the actual folder.
  4. It will prompt you to copy or link the file into the folder, you will almost always want to copy.
  5. It will ask you to overwrite the old image, we will want to do this as well.

Adjusting Plant Models

If you would like to adjust the order of the plants, or you would like to use the same texture for multiple stages you will want to change the model json files for the digi_plants

  1. Navigate to MDKExample/src/main/resources/assets.dm/models/block.
  2. digiplant.json (with no number at the end) is what the plant looks like while in your hand.
    3. { 
   "parent": "block/crop",   
   "textures": { 
       "crop": "dm:blocks/digi_plant0" 
   } 
}
  3. You will need to replace the text after the "dm:blocks/" to match what texture name you want to use for the look of the plant when it is being held.
  4. You can do this for all 8 other digi_plant#.json files. digi_plant0.json is the youngest plant, and digi_plant8.json is the oldest the plant will appear.
