Naming Items and Blocks

Naming items, blocks and creative tabs can be done to change the appearance of the name input the game of minecraft. It will not change how we reference the name in our coding environment.

Editing the Language File

  1. Locate the Language file by navigating to and opening MDKExample/src/main/resources/assets.dm/en_us.lang
  2. Find the name of the item you are replacing. If you are replacing a Sword, find 
    item.digi_sword.name
  3. Replace everything after the = sign with what you would like to have displayed in the game

Creating a new item or block name

  1. Navigate to and open MDKExample/src/main/resources/assets.dm/lang/en_us.lang
  2. Add in a new line of code at the bottom
  3. type "item.my_item.name= My New Item" for an item
  4. type "tile.my_block.name= My New Block" for a block
  5. Everything after the = sign in this line of code will show up as the name of this item in minecraft
  6. This is only the display name, we will still use my_item as name we refer to the item as in our code
    7.   item.my_item.name= My New Item
  tile.my_block.name= My New Block
