Introduction

Activity

Find DigiEvents

  1. Navigate to src/main/java/com/digitaladventures/digimod/DigiEvents.java
  2. Start by subscribing a new event within this class file. When we subscribe to an event it allows this code to listen for and trigger once the event associated with it occurs.

Create Super Sprinting Event

  1. Next create our new listening event called SuperSprinting, which will be passing through the PlayerTickEvent that we can give an instance/temporary name of 'event'
  2. event will now hold information about the player every tick of the game.
    3.     @SubscribeEvent
    public void SuperSprinting(PlayerTickEvent event){
    }
  3. Next we want to check if the player is sprinting
  4. If the player is sprinting then we will set the players walkspeed to 1F, which is about 1block per tick.
  5. If we are not sprinting we will se the speed back to 0.1F which is about the normal speed of a player in minecraft.

Completed Code

  @SubscribeEvent
	public void SuperSprinting(PlayerTickEvent event){
		if(event.player.isSprinting()) {
			event.player.capabilities.setPlayerWalkSpeed(1f);
		}else {
			event.player.capabilities.setPlayerWalkSpeed(0.1f);
		}
	}
