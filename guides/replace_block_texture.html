Replace Block Texture

This is a guide that will help students replace DigiMod items textures with their own designs which is the quickest way to have their creations come to life in minecraft.

Replace Block Texture File

  1. Name the file to correctly match the name of the file in the DigiMod you wish to replace.
  2. Navigate to MDKExample/src/main/resources/assets.dm/textures/blocks.
  3. Drag your block texture into the blocks folder either on top of any of the files in the folder, or on the actual folder.
  4. It will prompt you to copy or link the file into the folder, you will almost always want to copy.
  5. It will ask you to overwrite the old image, we will want to do this as well.
