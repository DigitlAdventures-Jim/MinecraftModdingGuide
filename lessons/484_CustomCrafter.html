484 Custom Crafter

  • Minecraft Modding 1.12
  • Digital Hackers 1st-8th
  • Skill Level Basic

Introduction

In this project we will be creating our own crafting recipes using JSON files. JSON files are used often with programming to send data in an organized way.

Activity

Students should draw a 3/3 grid (similar to tic tac toe), and then write ingredients into them to create a new recipe for minecraft.

Create a Shaped Recipe

Using the Create recipe guide walk through the creation of a custom shaped recipe.

You can use this json code below as an example of how to turn dirt into diamonds.

The "data": key is needed for many blocks or items which use meta data like wool or dyes. Using 32767 for the data value will let you use any type of wool or dye.


  {
      "type": "minecraft:crafting_shaped",
      "pattern": [
          "DDD",
          "DDD",
          "DDD"
      ],
      "key": {
          "#": {
              "item": "minecraft:dirt",
              "data":0
          }
      },
      "result": {
          "item": "minecraft:diamond",
          "count": 1
      }
  }
