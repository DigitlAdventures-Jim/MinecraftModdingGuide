480 Custom Armor

  • Minecraft Modding 1.12
  • Digital Hackers 1st-8th
  • Skill Level Basic

Introduction

In this project we will be designing custom armor using a model texturing program and then changing attributes for the armor.

Activity

Students should write down ratings for their favorite hero or role model. They should rate their strength, intelligence and kindness from 1-10. Students can share their answers with each other.

Create Armor Texture

Use the Create Armor Texture Guide to create a new armor. If designs are completed quickly you can also design the 4 16x16px Items that represent the armor.

Replace digi_layer_1.png and digi_layer_2.png Texture

Use the Replace Armor Guide to replace the digi_armor.

You should make sure to replace the digi armor names in the en_us.lang file as well.

Intermediate Challenge

Replace the Items that represent the armor (digi_helmet, digi_leggings, digi_chestplate, digi_boots) with your own.

Test your project

Make sure the amor can be worn and displays correctly. fn+f5 will change the players view from 1st person.

Advanced Challenge

As an advanced challenge try adjusting the values for the armor.

  1. Navigate and open the file /MDKExample/src/main/java/com/digitaladventures/digimod/init/DigiItems.java
  2. Your armor material is created at the line of code that looks like
    3. public static final ArmorMaterial ARMOR_MATERIAL_DIGI = EnumHelper.addArmorMaterial("armor_material_digi", Reference.MOD_ID + ":digi", 14,
			new int[] {2, 5, 7, 3}, 10, SoundEvents.ITEM_ARMOR_EQUIP_DIAMOND, 0.0F);
  3. This is armor material for the Digi Armor we are replacing.
  4. addArmorMaterial() is the key method that we are going to be using to customize our armor stats. The parameters are ordered and explained below.
    • String name : This is the material name Do not change this
    • String textureName : This is the name of file location of our armor set textures Do not change this
    • int durability : This is how long the armor will last when hit by enemies
    • int[] reductionAmounts : This is a list of the protection each armor provides helmet,chestplate,leggings and then boots
    • int enchantability : The chances you'll recieve great enchantments for your armor
    • SoundEvent soundOnEquip : The sound your armor makes when you put it on. Must use a "SoundEvents.SOUNDEVENTNAME"
    • float toughness : The amount of damage deflected before armor protection occurs. (Like an added shell to your whole armor set.)
