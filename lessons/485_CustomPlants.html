485 Custom Plants

  • Minecraft Modding 1.12
  • Digital Hackers 5th-8th
  • Skill Level Intermediate

Introduction

In this project we will be creating our own custom plant by replacing and adjusting the digi_plant. This will involve the changing of up to 7 plant model textures, a seed item and optionally a dropped item.

Activity

Create a very simple animation in piskel app to make it looke like a ball or dot is moving. Show students how to use the onion skin option so that they can view previous frames.

Create 7 Plant Model Textures and Seed

Using Piskel app and the .zip file export, export an animation of your plant growing, starting with a very tiny plant to one that fills up the animation frame.

You can use the Create Plant Texture Guide for steps on designing the model textures and seed texture.

Replace Plant and Seed Textures

You can use the Replace Plant Texture Guide for steps on replacing the plant and seeds as well as working with the plant model files and blockstates if necesarry.

This would be a good time to test your mod.

Changing Plant Drops

In minecraft once a plant grows to the last stage you can collect the drop. a quick way to grow a plant is by right clicking it with bonemeal.

  1. Navigate to and open /MDKExample/src/main/java/com/digitaladventures/digimod/blocks/DigiPlant.java
  2. Since a plant is a block, you can adjust or add to some of it's properties like hardess and light opacity.
  3. Make changes to the code below so that the return is the item you wish your plant to drop.
    4.     @Override
	     protected Item getCrop() {
		        return DigiItems.DIGI_FOOD;
	    }

Advanced: Create new drop Item

Use the Create Item Guide to design a new item that your plant should drop.
