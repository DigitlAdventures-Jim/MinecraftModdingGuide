Create Additional Workspaces (automatic creation)

Download the application to create student workspaces here. You'll only need one per work station.

Use the CreateStudentWorkspace app by dragging this application to the folder you want to create additional workspaces, ideally inside of a Minecraft_Mods folder on the dekstop

Double click on the app, type in the workspace name "StudentName_MinecraftMod", You can also create workspaces for more general purposes like a mod for a week of camp, or a birthday party.

Create Additional Workspace (manual creation)

If Automatic Creation does not work you may need to manually create a modding workspace by unzipping a forge-1.12.2 zipped folder from inside a student workspace folder.

Rename the folder before any other actions, otherwise the workspace will fail.

In terminal, type "cd" then drag in the folder to the terminal window.

Now that we are within the workspace folder you can run the command "./gradlew setupDecompWorkspace".

After this command completes successfully you can run "./gradlew eclipse".

Lastly you will need to replace the src folder within this workspace with a DigiMod src folder.
