DigiOre Class

The class is a file that holds all of the information for how our Block should act.

  • Navigate to the Java folder to find and open java/com/digitaladventures/digimod/blocks/DigiOre.java
  • This DigiOre class is extending another class or file called BlockBase which has all the code needed for building a basic block.
  • Extending is much better than recoding our entire block because we can use code already created by minecraft programmers.
  • The super() method tells this block to use the same constructor as the Base Block. It requires the name, and block material.

Change Ore Properties

Next we are going to be adjusting the properties of our block. We are going to do this by running some methods inside of the constructor of our Blocks class

A method is a task an object will do when we type it in. A constructor is a section of code that will run once at the creation of a class (it has the same name of the class in Java).

The methods inside our constructor have green slashes next to them. This means that they are "commented out". Commenting with two slashes means that the words after are to be ignored by the program and minecraft. This is how you can write notes in your code. This is also a great way to disable or enable code quickly.

  • setSoundType(SoundType.METAL); This method requires a SoundType, you can see the SoundType options by deleting ".METAL" and thentyping in "."
  • setHardness(5.0F); This method is how hard the block is to break with your player, This requires a number which ends with F for Float. Floats are decimal point numbers.
  • setResistance(15.0F); This sets how resistant your block is to explosions.
  • setHarvestLevel("pickaxe", 2); This sets the level of pickaxe needed to get the item from this block.

Change Drop Item

Change the return value of the getItemDropped method to a minecraft item using Items.ITEMNAMEHERE instead of DigiItems.ITEMNAMEHERE

If you'd like to drop a block you can type " Item.getItemFromBlock(Blocks.BEDROCK) "

  @Override
	public Item getItemDropped(IBlockState state, Random rand, int fortune) {
		return DigiItems.DIGI_GEM;
	}

Change the Quantity Dropped

Try to read the quantityDropped method, you can adjust the minimum and maximum numbers to adjust the random amount minimum and maximum.

  @Override
	public int quantityDropped(Random rand) {
		int max = 4;
		int min = 1;
		return rand.nextInt(max) + min;
	}
