Create Block Texture

Go To Piskel App

Go to www.piskelapp.com/create. Piskel app will allow us to create png images which are needed for minecraft textures. png files have transparent backgrounds which are necesarry for designing in games so that we don't have big white background boxes thoughout.

Resize Image

  1. Click on the resize button on the right side of the screen
  2. Change the size to 16x16 to match minecrafts pixel size
  3. Click the yellow resize button

Design Image

A standard block should have the entire design filled in. If you are looking to create custom block models then you can leave out some space, but you'll need to write a few Java functions, or adjust the model json file to ensure that the rendering of the box is correct.

  • Hover over the toolbar on the left to read what each tool does.
  • You will mostly just need the pen and eraser tools
  • For a smaller or larger pen you can click the boxes above the toolbar
  • To change the color you will need to click on the black box at the bottom and adjust both the hue (rainbow slider on the right) the saturation and brightness(the big box on the left)

Export Image

  1. On the right side toolbar click on the export button.
  2. Make sure to select the png tab
  3. Click the bottom download button. The top download button will also work if only 1 frame was drawn.
  4. Have students save the file to the desktop or wherever is easiest to find the file
