Create Plant Texture

A plant is a complex feature of minecraft as it includes an item and a block. The item is a seed we can plant and the Block is a BlockCrops that has up to 7 images.

Go To Piskel App

Go to www.piskelapp.com/create. Piskel app will allow us to create png images which are needed for minecraft textures. png files have transparent backgrounds which are necesarry for designing in games so that we don't have big white background boxes thoughout.

Resize Image

  1. Click on the resize button on the right side of the screen
  2. Change the size to 16x16 to match minecrafts pixel size
  3. Click the yellow resize button

Design Plant Block Images

Plants should be designed with from 4-8 drawings. Each plant is capable of 7 steps during it's growth but we can use the same design for two steps. Piskel app allows for animation frames on the left side which will allow for use to see the changes between each plant for a more smooth plant growth.

  • Make sure to enable the onion skin on the right, belwo the preview so we can compare our current frame to the previous frame.
  • You can create more frames by clicking add a new frame at the top left.
  • Hover over the toolbar on the left to read what each tool does.
  • You will mostly just need the pen and eraser tools
  • For a smaller or larger pen you can click the boxes above the toolbar
  • To change the color you will need to click on the black box at the bottom and adjust both the hue (rainbow slider on the right) the saturation and brightness(the big box on the left)

Export Image

  1. On the right side toolbar click on the export button.
  2. Make sure to select the zip tab
  3. Click the bottom download zip button. Change the prefix to my_plant_
  4. Have students save the file to the desktop or wherever is easiest to find the files
