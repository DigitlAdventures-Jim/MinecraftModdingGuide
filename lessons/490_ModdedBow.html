Add new Abilities

Navigate to and open src/main/java/com/digitaladventures/digimod/items/ItemDigiBow.java

This class extends the standard minecraft bow so that it acts just like it. We will want to add a method that when it is seen that this item is not enchanted, we will add enchantments.

  1. First you can navigate to src/main/java/com/digitaladventures/digimod/items/base/ItemBase.java and grab the onUpdate method which is similar to a forever loop, or every tick in other programs. This method will run as long as the item is in our inventory.
  2. Next we will need to check if the item is already enchanted so that we don't spam enchantments onto the bow.
  3. we can use 
    if(!stack.isItemEnchanted()){}
    , the exclamation point is like saying "is not" in front of a boolean or true/false data type.
  4. Next we can add in the enchantment adding script. 
    stack.addEnchantment(Enchantment, Level);
    5. The Enchantment required two arguments, the enchantment type and level. You can grab enchantments by accessing the Enchantmens class.
  5. It should look something like this. 
    stack.addEnchantment(Enchantments.FLAME, 3);
  6. Below is a list of enchantments you can use, some of them don't really utilize a level.
  • Vanishing Curse: Cursed item will disappear after player dies
  • Flame: Turns arrows into flaming arrows
  • Infinity: Shoots an infinite amount of arrows
  • Mending: Uses xp to mend your tools, weapons and armor
  • Power: Increases damage dealt by bow
  • Punch: Increases knockback dealt (enemies repel backwards)
  • Unbreaking:Increases durability of item

Finished Code

This should be located inside of the ItemDigiBow class

  public void onUpdate(ItemStack stack, World worldIn, Entity entityIn, int itemSlot, boolean isSelected)
  {
    if(!stack.isItemEnchanted()) {
       stack.addEnchantment(Enchantments.FLAME, 3);
    }
  }
